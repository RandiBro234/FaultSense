Cascade Classification Pipeline

Binary Detection

Failure Classification
Jika terdeteksi Failure, model kedua mengidentifikasi jenis kegagalan: TWF, HDF, PWF, OSF, atau RNF.
Smart Recommendation
Setiap prediksi disertai rekomendasi tindakan spesifik berdasarkan jenis kegagalan yang terdeteksi.
