## 📊 Technical Specifications

### Color Palette
```css
/* Primary */
#0A0A0A  /* Black - Background utama */
#22C55E  /* Green - Accent color */
#16A34A  /* Green Hover - Hover states */
#FFFFFF  /* White - Text on dark */

/* Text Colors */
#F9FAFB  /* Body text on dark */
#D1D5DB  /* Labels, secondary text */
#9CA3AF  /* Helper text, muted */
#374151  /* Text on light cards */

/* Transparencies */
rgba(34, 197, 94, 0.3)   /* Selection */
rgba(34, 197, 94, 0.2)   /* Borders */
rgba(34, 197, 94, 0.15)  /* Shadows */
rgba(34, 197, 94, 0.1)   /* Hover bg, focus ring */
```

### Typography
```css
font-family: 'Poppins', sans-serif;
font-weight: 400 (normal), 500 (medium), 600 (semibold), 700 (bold)
```

### Animations
```css
transition: all 0.3s ease;           /* Standard */
transition: background-color 0.2s ease;  /* Fast hover */
transition: color 0.3s ease;         /* Text color */
transition: opacity 0.3s ease;       /* Fade */
```

### Shadows
```css
/* Hover shadows */
box-shadow: 0 10px 25px rgba(34, 197, 94, 0.15);

/* Button hover */
box-shadow: 0 4px 12px rgba(34, 197, 94, 0.3);

/* Focus ring */
box-shadow: 0 0 0 3px rgba(34, 197, 94, 0.1);
```
